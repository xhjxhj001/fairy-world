# 快速开始指南

## 立即开始游戏

1. **打开游戏**
   - 直接在浏览器中打开 `index.html` 文件
   - 或者使用本地服务器（推荐）

2. **使用本地服务器（推荐）**
   ```bash
   # 在项目目录下运行
   python -m http.server 8000
   # 然后访问 http://localhost:8000
   ```

## 游戏流程

### 第一步：收集资源
- **白天**：点击花园中金色的资源田收集阳光露珠 ☀️
- **夜晚**：点击花园中紫色的资源田收集星光 ✨
- 资源也会自动生成（每10秒1点）

### 第二步：准备行囊
1. 点击底部"行囊"按钮 🎒
2. 选择食物、玩具或护身符（最多3件）
3. 点击"确认出发"

### 第三步：等待旅行
- 主角会自主外出旅行（30-90秒）
- 旅行期间可能会收到访客的礼物 🎁
- 主角归来时会带回照片和纪念品

### 第四步：查看收藏
- 点击"收藏"按钮 📷 查看收集的照片和纪念品
- 在"梦境"标签中查看睡眠模式后的梦境记录

## 特色功能

### 🌙 昼夜系统
- 游戏会根据系统时间自动切换白天/夜晚模式
- 夜晚模式（19:00-7:00）有特殊的视觉效果
- 夜晚可以进入睡眠模式

### 💫 社交功能
- **共享地图**：在社区中分享发现的神秘地点
- **访客记录**：查看来过你家的访客
- 输入地点代码（如 FOREST-001）可以分享该地点

### 🌙 助眠功能
- **晚安故事**：夜晚时聆听温馨故事
- **睡眠模式**：全屏睡眠模式，适合睡前使用
- 使用睡眠模式后，第二天会生成梦境记录

## 游戏提示

1. **资源收集**：白天只能收集阳光露珠，夜晚只能收集星光
2. **旅行时间**：旅行时间随机，耐心等待主角归来
3. **物品选择**：不同物品会影响旅行倾向，多尝试不同组合
4. **数据保存**：游戏数据自动保存，刷新页面不会丢失进度

## 常见问题

**Q: 主角一直不回来？**
A: 旅行时间最长90秒，请耐心等待。如果超过时间，刷新页面会自动恢复状态。

**Q: 如何获得更多资源？**
A: 点击资源田收集，或者等待自动生成。访客也会留下礼物。

**Q: 可以同时选择多个同类物品吗？**
A: 可以，只要总数量不超过3件即可。

**Q: 数据会丢失吗？**
A: 游戏数据保存在浏览器本地存储中，除非清除浏览器数据，否则不会丢失。
