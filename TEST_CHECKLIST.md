## 基础功能测试

### ✅ 1. 页面加载
- [ ] 页面正常加载，无控制台错误
- [ ] 游戏界面完整显示（状态栏、主场景、操作栏）
- [ ] 主角（🦊）正常显示在场景中
- [ ] 根据当前时间显示正确的昼夜模式

### ✅ 2. 资源系统
- [ ] 顶部状态栏显示阳光露珠和星光数量（初始为0）
- [ ] 点击阳光资源田（白天）可以收集阳光露珠
- [ ] 点击星光资源田（夜晚）可以收集星光
- [ ] 在错误时间点击资源田会显示提示信息
- [ ] 资源每10秒自动增加1点（需要等待验证）

### ✅ 3. 行囊系统
- [ ] 点击"行囊"按钮可以打开行囊面板
- [ ] 面板显示三个分类：食物、玩具、护身符
- [ ] 每个物品显示图标、名称和价格
- [ ] 点击物品可以选中/取消选中（最多3件）
- [ ] 资源不足时点击物品会提示"资源不足"
- [ ] 已选中的物品显示在"已选择"区域
- [ ] 点击"确认出发"按钮可以开始旅行

- [ ] 点击关闭按钮或遮罩层可以关闭面板

### ✅ 4. 旅行系统

- [ ] 主角有旅行动画效果（.traveling类）
- [ ] 资源被正确扣除
- [ ] 旅行中无法再次出发（显示提示）
- [ ] 旅行结束后主角状态变为"回来了！"
- [ ] 旅行结束后带回一张照片
- [ ] 有30%概率带回纪念品
- [ ] 旅行时间在30-90秒之间

### ✅ 5. 收藏系统
- [ ] 点击"收藏"按钮可以打开收藏面板

## 测试建议

1. **完整流程测试**：
   - 收集资源 → 购买物品 → 准备行囊 → 开始旅行 → 等待归来 → 查看收藏

2. **时间相关测试**：
   - 测试白天和夜晚的不同功能
   - 等待自动资源生成
   - 测试旅行时间恢复

3. **数据持久化测试**：
   - 进行一些操作后刷新页面
   - 检查数据是否正确恢复

4. **浏览器控制台检查**：
   - 打开开发者工具（F12）
   - 检查是否有JavaScript错误
   - 检查localStorage中的数据
